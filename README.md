### Getting started
To run the all locally :-

- clone the repo `git clone git@github.com:sgatana/su-fsd.git`
- run `npm i` or `npm install` (you also use yarn, pnpm or bun) to install packages
- Run below command to start development server


```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
